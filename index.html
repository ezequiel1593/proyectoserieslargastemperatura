Proyecto de construcción de series largas de temperatura de Argentina.

Este es un proyecto personal a largo plazo en el que se busca la construcción de series centenarias homogéneas (o lo más homogéneas posibles) de temperaturas medias mínimas y máximas mensuales, y por extensión estacionales y anuales.

Los pasos a seguir son los siguientes:

  • Búsqueda de datos y metadatos
  • Control de calidad de los datos
  • Imputación de datos faltantes
  • Homogeneización de la serie

Introducción

Primero es necesario hacer algunas definiciones. La temperatura mínima media mensual (Tnm) es el promedio de los datos diarios de temperatura mínima que componen el mes, siempre que la cantidad de datos faltantes no supere el 15%. En tal caso, la Tnm es igual a NA (missing value / dato faltante). La temperatura máxima media mensual (Txm) se define de manera análoga, pero utilizando datos diarios de temperatura máxima.

La temperatura mínima/máxima media estacional es el promedio de las Tnm/Txm que componen la estación, utilizando el criterio meteorológico/climatológico. La temperatura mínima/máxima media anual es el promedio de las Tnm/Txm que componen el año.
